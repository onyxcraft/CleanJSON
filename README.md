# CleanJSON

A powerful JSON formatter, validator, and viewer for macOS 14+.

## Features

- **JSON Formatting & Validation**
  - Auto-format/beautify JSON with configurable indentation (2 spaces, 4 spaces, or tabs)
  - Minify JSON to compact format
  - Real-time syntax validation with clear error messages and line numbers
  - Paste JSON directly or open from files

- **Tree View**
  - Collapsible tree visualization of JSON structure
  - Navigate complex JSON hierarchies easily
  - View node types, keys, and values
  - Copy JSON paths to clipboard

- **Search Functionality**
  - Search within JSON by key or value
  - View all matching results with paths
  - Quick navigation to found items

- **JSON Path Display**
  - Shows complete path to any selected node
  - Easy copy to clipboard for reference

- **Clipboard Operations**
  - Copy formatted or minified JSON
  - Paste JSON from clipboard
  - Quick access to common operations

- **JSON Comparison**
  - Compare two JSON files side by side
  - Diff highlighting shows added, removed, and modified values
  - Clear visualization of differences

- **YAML Conversion**
  - Preview JSON as YAML format
  - Copy YAML output
  - Side-by-side JSON/YAML view

- **Native macOS Integration**
  - Full dark mode support
  - Menu bar quick access with keyboard shortcuts
  - Native file open/save dialogs
  - Sandboxed for security

## System Requirements

- macOS 14.0 or later
- Compatible with both Intel and Apple Silicon Macs

## Installation

1. Download CleanJSON from the Mac App Store
2. Launch the app
3. Start formatting and validating JSON

## Usage

### Basic JSON Formatting

1. Paste JSON into the editor or click "Open" to load a file
2. Click "Format" to beautify the JSON
3. Use the indent picker to choose your preferred indentation style
4. Click "Copy" to copy the formatted JSON to clipboard

### Tree View

1. Load valid JSON into the editor
2. Switch to "Tree View" tab
3. Click nodes to expand/collapse
4. Select a node to view its path and details
5. Click "Copy Path" to copy the JSON path

### Compare JSON Files

1. Switch to "Compare" tab
2. Load JSON into left and right panels
3. Click "Compare" to see differences
4. Review highlighted changes

### Search

1. Enter search term in the search field
2. Click "Search" or press Enter
3. View results in the search panel
4. Click results to see their location
